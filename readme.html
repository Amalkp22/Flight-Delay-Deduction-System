Overview

Background

The data powering this system is extracted from the Marketing Carrier On-Time Performance table of the TranStats "On-Time" database (beginning January 2018). The time period is encoded in the compressed filename — for example, XXX_XXXXX_2001_1 contains data for the first month of 2001.

Flight delays cause significant inconvenience to passengers, increase airline operating costs, and disrupt airport logistics. This system predicts whether a flight will be delayed (departure ≥ 15 minutes) using historical flight records and live weather data fetched at prediction time.

What's New

✨ New Features

New Live Weather Forecast Integration

Weather conditions are now fetched live at prediction time using the Open-Meteo API — completely free, no account or API key required. Weather data is retrieved for both the origin and destination airports simultaneously, then converted into 25 structured model features that feed directly into delay probability estimation.

🌐
Zero-config API: Calls api.open-meteo.com/v1/forecast with the airport's latitude/longitude. No signup, no rate-limit headaches. Supported for all 30 major US airports in the built-in coordinate table.
📡
Live vs. Offline badges: Each airport weather card shows a green LIVE badge when data was freshly fetched, or a red OFFLINE badge with safe fallback defaults when the API is unreachable (timeout, unknown airport, HTTP error).
🧮
Weather risk score (0–10): A composite route risk score is computed from wind speed, visibility, precipitation, and weather condition at both airports. Displayed as a coloured badge (green / amber / red) beside every prediction.
⚡
25 engineered weather features are appended to each prediction request: temperature, humidity, wind speed & severity, visibility category, precipitation category, cloud cover, pressure, WMO condition code, temperature differential between airports, worst-leg visibility, and the combined route risk score.

WMO Weather Condition Codes Supported

☀️ Clear sky 🌤️ Mainly clear ⛅ Partly cloudy ☁️ Overcast 🌫️ Fog / Icy fog 🌦️ Drizzle (light/moderate/dense) 🌧️ Rain (slight/moderate/heavy) ❄️ Snow / Snow grains 🌧️ Showers ❄️ Snow showers ⛈️ Thunderstorm ± hail

Fallback behaviour: If the API call fails for any reason (timeout, unknown airport IATA code, HTTP error), _default_weather() returns safe neutral values (20 °C, Clear, 10 km vis, 5 m/s wind) so prediction always completes without crashing the app.

New Interactive Route Map

The Predict a Flight page now renders a live Folium map (via streamlit-folium) showing the full origin-to-destination route alongside current weather at each airport.

🛫
Airport markers with weather popups: Clicking either airport pin opens a popup showing live temperature, wind, visibility, humidity, and the weather icon for current conditions.
🎨
Risk-coloured flight path: The polyline connecting origin to destination is drawn in red for high weather risk (score > 6), orange for moderate (3–6), or green for low risk (< 3).
🔢
Risk score caption: Below the map a caption shows the route name, line colour meaning, and the numeric weather risk score out of 10 at a glance.
📐
Auto-fit bounds: The map automatically zooms and pans to fit both airport locations with comfortable padding, regardless of route distance.
🔌
Optional dependency: The map renders only when folium and streamlit-folium are installed. If missing, the app shows a helpful install prompt and continues working without the map.
Install map dependencies:
pip install folium streamlit-folium

Machine Learning

ML Models

🌲 Random Forest
Ensemble of 100 decision trees with balanced class weights. Best for feature importance analysis.
📈 Gradient Boosting
Sequential boosting with learning rate 0.1. Typically highest AUC-ROC score.
📐 Logistic Regression
Linear baseline with StandardScaler pipeline. Fast to train, interpretable coefficients.

Features Used

Month, Day of Week, Day of Month, Scheduled departure time, Distance, Scheduled elapsed time, Airline (label-encoded), Origin (label-encoded), Destination (label-encoded), Weather condition (encoded). New Plus 25 live weather features when weather data is available.

Target Variable

IS_DELAYED — 1 if departure delay ≥ 15 minutes, 0 otherwise.
